# Vue.js高仿饿了么外卖App 最火前端框架 1.0升级2.0

[视频教程地址](http://coding.imooc.com/class/74.html)

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# run unit tests
npm run unit

# run e2e tests
npm run e2e

# run all tests
npm test
```
