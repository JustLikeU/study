vertial-align文本类属性值

  1. text-top
    定义：
    盒子的顶部和父级content area的顶部对齐
  2. text-bottom
    定义：
    盒子的底部和父级content area的底部对齐


举个栗子：

img { vertical-align: text-top;}

18px 36px 54px
设置容器字体大小：
18px 36px 54px

并且：

1.元素vertical-align垂直对齐的位置与前后元素都没有关系

2.元素vertical-align垂直对齐的位置与行高没有关系，只与字体大小有关



实际应用：

兼容性较好，应用于表情图片（或原始尺寸背景图标）与文字的对齐效果

vertical-align实现上述应用

1.默认基线对齐

img { vertical-align: baseline;}

张三

缺点：图标偏上



2.顶线或者底线对齐

img { vertical-align: top;}
img { vertical-align: bottom;}

张三

缺点：虽然表面看似很完美，但是隐藏大坑的。因为它会受其他内联元素影响，造成巨大定位偏差



3.中线对齐

img { vertical-align: middle;}

张三

缺点：中线也是不错的选择，但是需要恰好的文字大小以及兼容性要求不高



4.文本底部对齐

img { vertical-align: text-bottom;}

张三

优点：使用文本底部较合适，不受行高以及其他内联元素影响
